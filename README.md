遙控無人機專業操作證 - 學科模擬測驗系統

這是一個基於 HTML5、CSS3 與純 JavaScript (Vanilla JS) 開發的單頁式模擬測驗應用程式。本專案將無人機學科題庫整合於單一檔案中，提供考生一個輕量、無需部署伺服器即可直接練習的測試環境。
✨ 功能特色

    🎯 隨機抽題：每次測驗系統會自動從內建題庫中隨機抽出 40 題進行測驗。

    🔀 選項洗牌：同一道題目每次出現時，其選項順序皆會隨機排列，防止考生死背答案位置。

    📊 自動計分：作答完畢送出後，系統會自動比對解答，並以百分制顯示最終得分與答對題數。

    🛡️ 防呆機制：若考生有題目漏未作答，點擊送出時會跳出提示視窗確認是否強制作答結束。

    🔄 無縫重測：提供「重新測驗」按鈕，一鍵清除畫面並重新抽取全新組合的考題。

    📱 響應式設計 (RWD)：簡潔的 UI 設計，無論使用電腦、平板或手機瀏覽器皆可順暢作答。

🚀 快速開始 (使用方式)

本專案為純前端網頁，無需安裝任何資料庫或伺服器環境。

    將原始碼複製並儲存為一個 HTML 檔案（例如：drone_quiz.html）。

    對著該檔案點擊右鍵，選擇使用瀏覽器開啟（支援 Chrome, Edge, Safari, Firefox 等現代瀏覽器）。

    網頁載入後即可直接開始測驗。

🛠️ 如何擴充或修改題庫

所有的題目皆儲存於程式碼中的 rawBank 陣列內。若您日後需要新增或修改題庫，只需使用文字編輯器（如記事本、VS Code）打開 HTML 檔案，並尋找 const rawBank = [...] 區段。

題庫資料格式說明：
JavaScript

{ 
    q: "遙控無人機活動期間，其所有人或操作人應負責任為何？",  // q: 代表問題內容 (Question)
    o: ["使用安全之責。", "風險管理之責。", "法規遵循之責。", "以上皆是。"], // o: 代表選項陣列 (Options)
    a: 3 // a: 代表正確答案的「索引值」(Answer Index)。索引從 0 開始算起，此例中 3 代表第四個選項「以上皆是。」
}

    新增題目：依照上述格式，將新題目加入 rawBank 陣列中，並用逗號 , 隔開即可。

    修改題數：若不想每次測驗 40 題，可於程式碼 initQuiz() 函式中尋找 .slice(0, 40)，將 40 修改為您期望的題目數量即可。

💻 技術堆疊

    HTML5: 負責網頁結構與表單。

    CSS3: 負責現代化的視覺呈現與互動回饋（Hover 效果）。

    JavaScript (ES6+): 負責題庫洗牌邏輯 (Fisher-Yates Shuffle 演算法)、DOM 渲染與計分邏輯。
